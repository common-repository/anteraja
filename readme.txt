Anteraja adalah plugin wordpress untuk pengguna woocommerce di Indonesia yang berfungsi untuk mengintegrasikan layanan ekspedisi anteraja dengan toko online berbasis woocommerce.

== Description == 

Halo Kak!, Plugin Anteraja adalah plugin wordpress untuk woocommerce yang berfungsi untuk mengintegrasikan ekspedisi Anteraja. Dengan menggunakan plugin ini, Kakak tidak hanya mendapatkan informasi ongkir Anteraja terintegrasi dengan cart toko online, namun juga dapat melakukan pemesanan kurir Anteraja secara online dan realtime dari dashboard toko online Kakak sehingga Kakak sebagai pemilik toko online dapat memperoleh resi otomatis dari Anteraja dan penjadwalan penjemputan paket dari kurir Satria. Anteraja adalah sebuah perusahaan ekspedisi pengiriman di Indonesia yang berkolaborasi dengan Epeken.com menyediakan plugin Anteraja untuk toko online. Untuk menggunakan plugin ini, Kakak perlu memiliki [licensi Epeken All Kurir](https://www.epeken.com/shop/epeken-all-kurir-license) untuk mendapatkan layanan dan dukungan teknis dari Epeken.com. Plugin [Epeken All Kurir](https://wordpress.org/plugins/epeken-all-kurir) wajib diinstal juga bersama dengan plugin Anteraja sebagai prasyaratnya. Untuk mendapatkan akses ke Anteraja, Kakak harus terdaftar sebagai pelanggan Anteraja. [Daftar di sini gratis!!](https://bisnis.anteraja.id/register/online-seller). 

== Support Marketplace ==

Plugin anteraja sudah support marketplace dengan plugin Dokan dan MultiVendorX. Plugin marketplace mainstream yang lain sedang dalam pengembangan untuk dapat didukung oleh plugin Anteraja. Tunggu saja di versi berikutnya, ya Kak.

== Installation == 

Seperti plugin wordpress pada umumnya, proses instalasi plugin ini dapat dilakukan dengan mengunduh plugin ini dari wordpress repository dan menginstallnya dari format zip, atau dapat menginstal langsung dari layar Administrator wordpress pada bagian Plugins. Sesuai pada bagian Deskripsi, plugin ini membutuhkan licensi [Epeken All Kurir](https://www.epeken.com/shop/epeken-all-kurir-license) dan wajib menginstalnya bersama dengan plugin [Epeken All Kurir](https://wordpress.org/plugins/epeken-all-kurir). Tanpa licensi dan plugin Epeken All Kurir plugin ini tidak berfungsi. Selain itu Kakak harus terdaftar sebagai pelanggan Anteraja. [Daftar di sini gratis!!](https://bisnis.anteraja.id/register/online-seller) atau dapat [Menghubungi Sales Anteraja](https://wa.me/6281284027676)

= Minimum Requirement =

* Wordpress 4.0 or greater
* Woocommerce version 3.0 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
* Epeken All Kurir version 1.1.8.6.26 or greater

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
